## ğŸ Getting Started

### Prerequisites

- **Node.js** (v18 or higher)
- **Bun** (v1.0 or higher)

### Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/ollie/finai.git
    cd finai
    ```

2.  **Install dependencies:**

    ```bash
    bun install
    ```

3.  **Start the development server:**

    ```bash
    bun run dev
    ```

    The application will be available at `http://localhost:3889`.
