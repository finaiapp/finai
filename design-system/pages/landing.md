# Landing Page Design System

> **Extends:** `MASTER.md`
> **Focus:** Engagement • Conversion • Trust

## 1. Layout & Structure

- **Hero Section:** Full viewport height (`min-h-screen`) or huge spacing (`py-24`).
- **Container:** `max-w-7xl` centered with `px-4 sm:px-6 lg:px-8`.
- **Spacing:** Large vertical rhythm. Use `gap-16` or `gap-24` between sections.

## 2. Component Overrides

### Typography
- **Headings:** Use `Outfit` font with `font-bold` weight.
- **Hero Title:** `text-5xl md:text-7xl tracking-tight`.
- **Section Title:** `text-3xl md:text-4xl font-semibold text-center mb-12`.

### Buttons (CTAs)
- **Primary CTA:** Larger padding (`px-8 py-4`), `text-lg font-semibold`, `rounded-full` or `rounded-xl`.
- **Shadow:** Stronger shadow (`shadow-lg hover:shadow-xl`) and glow effect (`shadow-primary-500/20`).

## 3. Visual Assets
- **Imagery:** High-quality device mockups with consistent shadows.
- **Backgrounds:** Subtle gradients (`bg-gradient-to-br from-slate-50 to-white`) or mesh gradients.
- **Abstract Shapes:** Use rounded geometric shapes in background with low opacity for depth.

## 4. Trust Barriers (Social Proof)
- **Logos:** Grayscale monochrome with `opacity-50 hover:opacity-100 transition-opacity`.
- **Testimonials:** Cards with subtle border `border-slate-100` and generous padding.
